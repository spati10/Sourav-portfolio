Portfolio Website
This is a personal portfolio website project showcasing my skills, projects, and contact information. It includes HTML pages with sections for About, Projects, Social links, Contact form, and CV download.

Features: 
Responsive multi-page HTML structure

Contact form integrated with Web3Forms for email submissions

Downloadable CV (PDF) link

Sections for personal info, project portfolio, and social links

Project Structure:
PORTWEBSITE/
├── .git/                            # Git repository data
├── assets/                         # Static resources like CSS, JS, images, fonts
│   ├── css/
│   │   ├── fontawesome-all.min.css
│   │   ├── main.css                # Main stylesheet
│   │   └── noscript.css
│   ├── js/                        # JavaScript files (if any)
│   ├── sass/                      # SASS source files (if used)
│   └── webfonts/                  # Font files
├── images/                        # Image assets used in the site
├── elements.html                 # HTML elements or templates page
├── index.html                   # Main homepage of the portfolio
├── LICENSE.txt                   # License information for the project
├── README.txt                   # This README file
└── Sourav_Web CV.pdf            # Your CV file available for download

Setup & Usage:

Clone the repo locally

Open index.html in a web browser to view the portfolio

Edit HTML/CSS files in the Components folder to customize content and style

To receive messages, the contact form uses Web3Forms – replace the access key with your own

Upload your CV PDF to the assets folder and update the link in the About page
